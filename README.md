#Cyber security incident triage pack- healthcare phishing scenario 

##Overview 
-
this is a beginner cybrsecurity portfolio project created to demonstrate basic incident logging, triage, escalation, and awareness of common cyber threats sffecting healthcare organisations.
The project is based on a fictional phishing incident in a healthcare environment. It was created to support my development for trainee cybersecurity roles.

#Project Objectives
-
 -Demonstrate basic incident documentation skills 
 -Show understanding of incident triage and esclation 
 -Summarise common cyber threats relevant to healthcare organisations
 -Present information in a clear and professional format

 #Contents
 - 
 this project includes
 - a mock incident report log
 - a triage and esclation note
 - a short threat summary covering phishing, ransomware, and third-party compromise

##Scenario 
-
A staff member in the finance department receives a phishing email disguised as a Microsoft 365 password reset request. After clicking the link and entering their credentials, suspicious login attempts are detected fram an overseas IP address. The incident is logged, triaged and esclated for investigation.
